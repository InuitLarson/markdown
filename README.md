# MARKDOWN
Application Markdown avec Node.js

## installation

Une seule fois :

	sudo npm install

Lancement du serveur Node.js

	npm run dev
	
dans un navigateur :
	
	http://localhost:8000
	
## Stopper Node.js 
	ps aux | grep node
	
Déterminer l'ID du processus (deuxième à partir de la gauche) :

	kill -9 PROCESS_ID
